核心觀念秘笈

【觀念一】公差與第 $n$ 項公式

1. 公差 (d)：後項減前項的差。即 d = a2 - a1。
2. 第 n 項公式：an = a1 + (n - 1) × d。
(記法：要走到第 n 項，必須從第 1 項出發，跨出 n-1 個步伐)

【觀念二】尋找項數 $n$

已知數值求它是第幾項，可以利用公式反推：
總距離 ÷ 步伐大小 + 1
公式：n =

an - a1
d
+ 1。

【觀念三】等差中項

若 a, b, c 三數成等差數列，則中間的 b 稱為 a 和 c 的等差中項。
性質：左右相加除以 2，就是中間的數。
公式：b =

a + c
2
（或 2b = a + c）
